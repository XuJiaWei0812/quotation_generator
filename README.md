# 報價單產生器 (Quotation Generator)

這是一個專為台灣自由接案者 (Freelancers) 設計的現代化報價單產生器。基於純前端技術 (HTML/CSS/JS) 構建，無需後端伺服器，資料完全儲存於您的瀏覽器中，安全且隱私。

🔗 **[線上使用 Live Demo]()**

## ✨ 主要功能

- **完全免費且開源**：無浮水印、無隱藏費用。
- **符合台灣商業標準**：包含統編、稅務 (5% 營業稅開關)、折扣欄位、銀行匯款資訊與簽名欄位。
- **多版本管理**：
  - 可建立無限多張報價單。
  - 支援版本切換與歷史紀錄查詢。
  - **智慧命名**：自動偵測並防止單號或專案名稱重複。
- **資料安全與備份**：
  - **Auto-Save**：輸入時自動儲存至瀏覽器 (Local Storage)，防止意外關閉遺失資料。
  - **JSON 備份/還原**：可匯出所有報價單紀錄，或匯入還原至另一台電腦。
- **優化使用者體驗**：
  - **響應式設計 (RWD)**：支援手機與電腦操作。
  - **轉場動畫**：切換版本時具備流暢的視覺效果。
  - **一鍵列印**：完美適配 A4 紙張格式，可直接另存為 PDF。

## 🛠️ 技術棧

- **HTML5 / CSS3 / JavaScript**
- **Bootstrap 5** (用於 UI 格線系統與元件)
- **Font Awesome** (圖示庫)
- **Local Storage API** (資料持久化)

## 🚀 如何使用

1.  **線上使用**：直接點擊上方 Demo 連結。
2.  **本地使用**：
    - 下載此專案 (Code -> Download ZIP)。
    - 解壓縮後，直接用瀏覽器打開 `index.html` 即可使用。

## 📂 檔案結構

- `index.html`: 應用程式主結構。
- `style.css`: 樣式表 (包含列印設定與動畫)。
- `script.js`: 核心邏輯 (資料管理、備份還原、DOM 操作)。

## 📝 注意事項

- 本工具使用瀏覽器 `Local Storage` 儲存資料，若您清除瀏覽器快取 (Cache)，資料將會消失。**請務必定期使用「備份 (JSON)」功能下載備份檔。**
- 本程式碼僅供輔助使用，正式合約建議諮詢法律專業人士。
